# QQbot-KunJinKao

## 简介
锟斤拷是一个辅助性的 QQ 聊天机器人。
* 基于 OICQ 框架
* 支持最低 Node 版本 14
* 项目开源于 [GitHub](https://github.com/gbCzz/QQBot-KunJinKao)

当前版本： ver 2.0.1
